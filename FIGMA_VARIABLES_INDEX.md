## 🔍 核心概念速查

### Variables 基础

```
Collection (集合)
  ├─ 包含一组相关变量
  ├─ 可以有多个 Mode（模式）
  └─ 例如：Primitives/Color

Mode (模式)
  ├─ 同一变量在不同场景的值
  ├─ 例如：Light / Dark
  └─ 切换 Mode 自动切换所有变量值

Variable (变量)
  ├─ 可复用的设计值
  ├─ 类型：Color, Number, String, Boolean
  └─ 可以引用其他变量
```

### 原始 vs. 语义

```
原始变量 (Primitive Variables)
├─ 定义：基础色值、数值
├─ 示例：brand/primary/500 = #3387FF
├─ 特点：具体值，不引用其他变量
└─ 用途：被语义变量引用

语义变量 (Semantic Variables)
├─ 定义：用途描述
├─ 示例：interactive/default → {brand/primary/500}
├─ 特点：引用原始变量
└─ 用途：直接应用到组件
```

### 映射关系

```
原始变量 → 语义变量 → 组件应用

brand/primary/500
    ↓ (引用)
interactive/default
    ↓ (应用)
Button Fill
```

## 💡 关键操作要点

### ✅ 必须做对的 5 件事

1. **原始变量不直接用于组件**
   ```
   ❌ Button Fill = {brand/primary/500}
   ✅ Button Fill = {interactive/default}
   ```

2. **语义变量必须映射原始变量**
   ```
   ❌ text/primary = #0F1721
   ✅ text/primary → {neutral/1000}
   ```

3. **深色模式必须重新映射**
   ```
   Light: text/primary → {neutral/1000}
   Dark:  text/primary → {neutral/0}
   ```

4. **命名使用斜杠分隔**
   ```
   ✅ text/primary
   ✅ brand/primary/500
   ❌ textPrimary
   ❌ brand-primary-500
   ```

5. **检查对比度**
   ```
   正文文本：≥ 4.5:1
   大文本：≥ 3:1
   UI 元素：≥ 3:1
   ```
