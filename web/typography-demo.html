使用建议

1. 标准场景

页面布局： h1 (32px) → h2 (24px) → body-lg (16px)

表单： label-md (14px) + body-md (14px) + caption-md (12px)

按钮： label-lg (16px) / label-md (14px) / label-sm (13px)

2. 响应式

Desktop (≥1024px)： 使用 Web 端规范（display/heading）

Tablet (768-1023px)： Display 缩小，Heading 保持或缩小一级

Mobile (<768px)： 切换到移动端规范（H1/H2/H3）

3. 无障碍

✅ 正文至少 16px（移动端）/ 14-16px（Web 端）

✅ 辅助文字至少 12px（可读性底线）

✅ 行高 1.5-1.8（舒适阅读）

✅ 文字与背景对比度至少 4.5:1（WCAG AA 标准）
